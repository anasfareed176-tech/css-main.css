🔧 Installation Guide

General Steps for Mojo Launcher:

  1. Download the mod .jar file from the official link below.
  2. Open Mojo Launcher and navigate to your instance's folder.
  3. Place the .jar into the mods/ folder.
  4. Make sure you have the correct mod loader installed (Fabric / Forge / NeoForge).
  5. Launch the game and check the mods menu to confirm it loaded.
